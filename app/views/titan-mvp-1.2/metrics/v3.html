Form metrics dashboard

Showing performance data for forms created and published using Form Designer.

Last updated: Thursday, 13 November 2025 at 12:35pm

This dashboard provides key performance indicators (KPIs) for forms created, published, and submitted through the Form Designer service. Metrics include quarterly trends for form creation, publication rates, submission volumes, and average time to publish.

These metrics are updated every month.

Headlines

Since 1 October 2024, there have been 42 forms created and 28 forms published in the Form Designer service.

Form creation and publication rates are following expected patterns. The total number of forms created this quarter (Oct to Dec 2024) is higher than the previous quarter, with an average of 16 days from creation to first publication.
